<p align="center" dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">
  Ionic 是一个开源应用程序开发工具包，用于使用 JavaScript 和 Web 从单个代码库构建现代、快速、高质量的跨平台本机和渐进式 Web 应用程序。
</font></font></p>
<p align="center" dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">
